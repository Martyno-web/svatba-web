# Svatební web

## Cíl projektu

Vytvořit jednoduchý, elegantní a responzivní jednostránkový svatební web.

Web bude hostovaný zdarma přes GitHub Pages.

Nejde pouze o vytvoření svatebního webu.

Projekt slouží zároveň k tomu, abych se naučil moderní HTML, CSS a JavaScript.

Při návrzích proto preferuj:
- jednoduchá řešení,
- čistou strukturu,
- srozumitelný kód,
- běžné webové standardy.

Pokud existuje více řešení, zvol to, které je nejvhodnější pro výuku začátečníka.

## Priorita

Projekt navrhuj metodou mobile-first.

Předpokládá se, že více než 90 % návštěvníků bude web používat na mobilním telefonu.

Desktop je sekundární zařízení.

## Požadavky

- Web musí být plně responzivní.
- Primárně optimalizuj pro šířky 360–430 px.
- Navigace musí být dobře ovladatelná prstem.
- Text musí být bez přibližování dobře čitelný.
- Tlačítka musí mít dostatečně velkou dotykovou plochu.
- Žádné vodorovné posouvání stránky.
- Optimalizuj obrázky pro rychlé načítání.
- Používej HTML5, moderní CSS a čistý JavaScript bez frameworků.
- Preferuj jednoduché, dobře udržovatelné řešení před složitými efekty.
